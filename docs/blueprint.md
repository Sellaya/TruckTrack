# **App Name**: TruckOps Tracker

## Core Features:

- Income Tracking: Log income from each delivery.
- Expense Tracking: Track fuel, maintenance, and other operating expenses, including multi-currency support.
- Profit Calculation: Calculate profit per trip, day, week, or month.
- Performance Dashboard: Visualize key metrics, such as revenue, expenses, and profit margins.
- Report Generation: Generate and export detailed reports for accounting.
- Trip Logging: Log all of the relevant information for each trip
- AI expense prediction: Predicts your operational expenses using AI as a helpful tool based on your input information.

## Style Guidelines:

- Primary color: Dark Blue (#2c3e50) for a professional and reliable feel.
- Background color: Light gray (#f0f0f0) to provide a clean, neutral backdrop.
- Accent color: Orange (#e67e22) to highlight key metrics and actions, creating visual interest.
- Body and headline font: 'Inter' (sans-serif) for clear readability and a modern aesthetic.
- Use clear, consistent icons to represent different income and expense categories.
- Design a clean, intuitive layout with clear data hierarchy for easy navigation.
- Use subtle transitions and animations to enhance user experience.
